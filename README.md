# Cycle Therapy NYC – Demo Website

## Overview
Cycle Therapy NYC is a modern and responsive demo business website created for a cycling-related brand. The project focuses on delivering a clean user interface, structured layout, and mobile-friendly experience.

## Features
- Responsive design for desktop, tablet, and mobile
- Modern and clean landing page UI
- Organized business and service sections
- Smooth navigation experience
- Professional website layout

## Tech Stack
- HTML
- CSS
- JavaScript
- Vercel Deployment

## Live Demo
🔗 https://cycletherapynyc.vercel.app

## Deployment
Deployed using Vercel for fast and reliable hosting.

## Purpose
This project was built as a demo webpage to practice frontend development, responsive web design, and creating professional business websites suitable for client work.
